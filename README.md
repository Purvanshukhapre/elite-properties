## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation
```bash
npm install
npm run dev
```

### Development
```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

### Production Build
```bash
npm run build
```

The build will be output to the `dist/` directory, ready for deployment to any static hosting service.

### Environment Variables
Create a `.env` file with:
```
VITE_API_URL=your_api_endpoint
VITE_GOOGLE_CLIENT_ID=your_google_client_id
```
